# Images pour la documentation

Ce répertoire contient les images utilisées dans les articles de documentation.

## Images attendues

- `longview-parametrage.png` (ou `.jpg`, `.jpeg`) : Capture d'écran de l'interface de paramétrage complète
- `longview-simulation.png` (ou `.jpg`, `.jpeg`) : Capture d'écran du graphique de simulation avec les percentiles 10, 50, 90%

## Format recommandé

- Format : PNG ou JPEG
- Résolution : Minimum 1200px de largeur pour une bonne qualité d'affichage
- Taille : Optimiser les images pour le web (compression raisonnable)
